## 1. 데일리 콘텐츠 예시 (레벨별)

### N5 (입문) - 기초 일상

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📚 오늘의 일본어 (N5 레벨)
2026.03.15 (월)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💬 오늘의 문장

  わたしは　まいあさ　七時(しちじ)に　おきます。
  저는 매일 아침 7시에 일어납니다.

📝 오늘의 단어 (3개)
  • 毎朝 (まいあさ) - 매일 아침
  • 起きる (おきる) - 일어나다
  • 七時 (しちじ) - 7시

💡 오늘의 문법
  • ~に おきます : 「~시에 일어납니다」
    시간 + に + 동사 → 특정 시간에 하는 행동

📖 추가 예문
  ① 六時(ろくじ)に おきます。 (6시에 일어납니다)
  ② 十時(じゅうじ)に ねます。 (10시에 잡니다)

[퀴즈 풀기] [단어 복습] [문법 더보기]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### N4 (초급) - 일상 회화

```

📚 오늘의 일본어 (N4 레벨)
2026.03.15 (월)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💬 오늘의 문장

  週末(しゅうまつ)、友達(ともだち)と映画(えいが)を
  見(み)に行(い)こうと思(おも)っています。
  주말에 친구와 영화를 보러 가려고 생각하고 있습니다.

📝 오늘의 단어 (4개)
  • 週末 (しゅうまつ) - 주말
  • 友達 (ともだち) - 친구
  • 映画 (えいが) - 영화
  • 思う (おもう) - 생각하다

💡 오늘의 문법
  • ~(よ)うと思っている : 「~하려고 생각하고 있다」
    의지형 + と思っている → 계획/의도를 나타냄
    ※ ~と思う (지금 결심) vs ~と思っている (이전부터의 계획)

📖 추가 예문
  ① 来年(らいねん)、日本(にほん)に行こうと思っています。
    (내년에 일본에 가려고 생각하고 있습니다)
  ② 新(あたら)しい仕事(しごと)を探(さが)そうと思っています。
    (새 일을 찾으려고 생각하고 있습니다)

[퀴즈 풀기] [단어 복습] [문법 더보기]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### N3 (중급) - NHK Easy 뉴스

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📚 오늘의 일본어 (N3 레벨)
2026.03.15 (월)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📰 NHK Easy 뉴스

  外国人(がいこくじん)の観光客(かんこうきゃく)が増(ふ)えて、
  京都(きょうと)のバスが混(こ)んでいます。京都市(きょうとし)は、
  観光客(かんこうきゃく)だけが乗(の)れる特別(とくべつ)な
  バスを走(はし)らせることにしました。料金(りょうきん)は
  一回(いっかい)700円(えん)で、有名(ゆうめい)な
  お寺(てら)や神社(じんじゃ)を回(まわ)ります。

  외국인 관광객이 늘어 교토 버스가 혼잡합니다. 교토시는
  관광객만 탈 수 있는 특별 버스를 운행하기로 했습니다.
  요금은 1회 700엔이며 유명한 절과 신사를 순회합니다.

📝 오늘의 단어 (5개)
  • 観光客 (かんこうきゃく) - 관광객
  • 混む (こむ) - 혼잡하다, 붐비다
  • 特別 (とくべつ) - 특별
  • 料金 (りょうきん) - 요금
  • 神社 (じんじゃ) - 신사

💡 오늘의 문법
  • ~ことにする : 「~하기로 하다」 (결정)
    동사 사전형 + ことにする → 스스로의 결정
    ※ ~ことになる (외부 결정) 와 구별

  • ~だけが : 「~만이」 (한정)
    명사 + だけが → 오직 ~만

📖 핵심 표현
  ① バスを走らせる → 버스를 달리게 하다 (사역)
  ② お寺や神社を回る → 절이나 신사를 돌다 (순회)

[퀴즈 풀기] [전문 읽기] [단어 복습]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### N2 (중상급) - 뉴스/칼럼

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📚 오늘의 일본어 (N2 레벨)
2026.03.15 (월)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📰 오늘의 뉴스

  人工知能（じんこうちのう）を活用（かつよう）した翻訳
  （ほんやく）技術（ぎじゅつ）の進歩（しんぽ）により、
  言語（げんご）の壁（かべ）が急速（きゅうそく）に
  低（ひく）くなりつつある。しかし、専門家（せんもんか）は
  「文化的（ぶんかてき）なニュアンスや文脈（ぶんみゃく）に
  依存（いぞん）する表現（ひょうげん）については、依然
  （いぜん）として人間（にんげん）の判断（はんだん）が
  不可欠（ふかけつ）だ」と指摘（してき）している。

  AI 번역 기술의 진보로 언어 장벽이 급속히 낮아지고 있다.
  그러나 전문가는 "문화적 뉘앙스나 문맥에 의존하는 표현은
  여전히 인간의 판단이 불가결하다"고 지적하고 있다.

📝 오늘의 단어 (7개)
  • 人工知能 (じんこうちのう) - 인공지능
  • 活用 (かつよう) - 활용
  • 進歩 (しんぽ) - 진보
  • 急速 (きゅうそく) - 급속
  • 文脈 (ぶんみゃく) - 문맥
  • 依然として (いぜんとして) - 여전히
  • 不可欠 (ふかけつ) - 불가결

💡 오늘의 문법 (2개)
  • ~つつある : 「~하고 있는 중이다」 (점진적 변화)
    동사 ます형 + つつある
    例: 低くなりつつある (낮아지고 있는 중이다)
    ※ ~ている보다 격식체, 서서히 진행되는 변화에 사용

  • ~として : 「~로서」
    명사 + として → 자격/입장을 나타냄
    例: 依然として (여전히) ← 부사적 용법

[퀴즈 풀기] [전문 읽기] [단어 복습]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```

📰 오늘의 칼럼 — 「働き方の再定義」

  かつて終身雇用（しゅうしんこよう）が当然視（とうぜんし）
  されていた日本社会（にほんしゃかい）において、働（はたら）き

  （かいきん）やリモートワークの普及（ふきゅう）を契機
  （けいき）に、個人（こじん）が主体的（しゅたいてき）に
  キャリアを構築（こうちく）するという意識（いしき）が
  浸透（しんとう）しつつある。

  とはいえ、こうした変革（へんかく）が全（すべ）ての
  労働者（ろうどうしゃ）に恩恵（おんけい）をもたらして
  いるとは言（い）い難（がた）い。非正規（ひせいき）雇用者
  （こようしゃ）にとっては、柔軟性（じゅうなんせい）の
  名（な）の下（もと）に雇用（こよう）の安定（あんてい）が
  損（そこ）なわれかねないという懸念（けねん）も根強
  （ねづよ）い。

📝 오늘의 단어 (10개)
  • 終身雇用 (しゅうしんこよう) - 종신고용
  • 不可逆的 (ふかぎゃくてき) - 불가역적
  • 潮流 (ちょうりゅう) - 조류, 추세
  • 契機 (けいき) - 계기
  • 浸透 (しんとう) - 침투, 확산
  • 変革 (へんかく) - 변혁
  • 恩恵 (おんけい) - 은혜, 혜택

  • 懸念 (けねん) - 우려

💡 오늘의 문법 (3개)
  • ~かねない : 「~할 수도 있다」 (부정적 가능성)

    例: 損なわれかねない (손상될 수도 있다)

    例: 恩恵をもたらしているとは言い難い

    名詞 + の名の下に → 표면적 이유/명분
    例: 柔軟性の名の下に (유연성이라는 명목 하에)

## 2. 퀴즈 유형별 예시

### 2.1 읽기 퀴즈 (후리가나 맞추기) — 전 레벨

```
━━━━━━━━━━━━━━━━━━━━━━━━━
📝 읽기 퀴즈 (N3)
━━━━━━━━━━━━━━━━━━━━━━━━━

Q. 「観光客」의 읽기는?

  A) かんこうきゃく
  B) かんこうかく
  C) みこうきゃく
  D) かんひかりきゃく

━━━━━━━━━━━━━━━━━━━━━━━━━
```

```
✅ 정답! A) かんこうきゃく

📖 해설
  観 (かん) = 볼 관
  光 (こう) = 빛 광
  客 (きゃく) = 손님 객

  💡 관련 단어:
  • 観光 (かんこう) - 관광
  • 観光地 (かんこうち) - 관광지
  • 旅行客 (りょこうきゃく) - 여행객
```

### 2.2 어휘 퀴즈 (4지선다) — 전 레벨

```
━━━━━━━━━━━━━━━━━━━━━━━━━
📝 어휘 퀴즈 (N4)
━━━━━━━━━━━━━━━━━━━━━━━━━

Q. 「混む」의 뜻은?

  A) 섞다
  B) 혼잡하다, 붐비다 ✦
  C) 포함하다
  D) 곤란하다

━━━━━━━━━━━━━━━━━━━━━━━━━
```

```
✅ 정답! B) 혼잡하다, 붐비다

📖 해설
  混む (こむ) = 붐비다, 혼잡하다

  💡 활용:
  • 電車が混んでいる (전철이 붐비고 있다)
  • 道が混む (길이 막히다)
  • 混み合う (こみあう) - 북적거리다

  ⚠️ 헷갈리기 쉬운 단어:
  • 混ぜる (まぜる) - 섞다
  • 込む (こむ) - 들어가다, 넣다
```

### 2.3 문법 빈칸 퀴즈 — 전 레벨

```
━━━━━━━━━━━━━━━━━━━━━━━━━
📝 문법 퀴즈 (N3)
━━━━━━━━━━━━━━━━━━━━━━━━━

Q. 다음 빈칸에 알맞은 것은?

  「日本に留学する（　　）にしました。」
   일본에 유학하기로 했습니다.

  A) もの
  B) こと
  C) ため
  D) ところ

━━━━━━━━━━━━━━━━━━━━━━━━━
```

```
✅ 정답! B) こと

📖 해설
  ~ことにする = ~하기로 하다 (자신의 결정)

  💡 비교:
  • ~ことにする → 자발적 결정
    来年結婚することにした (내년에 결혼하기로 했다)
  • ~ことになる → 외부적 결정/상황
    来月転勤することになった (다음 달 전근가게 됐다)

  ⚠️ 주의:
  ~ものにする는 존재하지 않는 표현!
```

### 2.4 문장 해석 퀴즈 — N4-N1

```
━━━━━━━━━━━━━━━━━━━━━━━━━
📝 해석 퀴즈 (N2)
━━━━━━━━━━━━━━━━━━━━━━━━━

Q. 다음 문장을 한국어로 해석하세요.

  「言語の壁が急速に低くなりつつある。」

  답을 입력해주세요 ✏️

━━━━━━━━━━━━━━━━━━━━━━━━━
```

```
사용자: 언어의 벽이 급속히 낮아지고 있다

✅ 정답!

📖 모범 해석
  "언어의 장벽이 급속히 낮아지고 있는 중이다."

  💡 채점 포인트:
  ✅ 言語の壁 → 언어의 벽/장벽
  ✅ 急速に → 급속히/빠르게
  ✅ 低くなる → 낮아지다
  ✅ ~つつある → ~하고 있다 / ~하는 중이다

  📌 ~つつある vs ~ている
  둘 다 "~하고 있다"지만, ~つつある는
  서서히 진행되는 변화에 사용하며 격식체입니다.
```

### 2.5 문장 만들기 퀴즈 — N4-N1

📝 작문 퀴즈 (N3)
━━━━━━━━━━━━━━━━━━━━━━━━━

Q. 다음 단어를 모두 사용하여 일본어 문장을 만드세요.

  📦 사용 단어: 週末 / 友達 / 映画 / 見る

  힌트: 「주말에 친구와 영화를 봤습니다」

  답을 입력해주세요 ✏️

━━━━━━━━━━━━━━━━━━━━━━━━━
```

```
사용자: 週末、友達と映画を見ました。

✅ 정답!

📖 모범 답안
  「週末、友達と映画を見ました。」
  (주말에 친구와 영화를 봤습니다.)

  💡 다른 올바른 표현:

  • を → 목적어 조사
  • ~ました → 과거 정중체

### 2.6 어순 배열 퀴즈 — N5-N3

```

```
